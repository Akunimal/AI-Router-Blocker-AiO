# AI Blocker 🚫🤖

AI Blocker is an open-source tool and configuration suite designed to block, filter, and manage AI crawlers, scrapers, and automated agents. It helps webmasters, developers, and content creators protect their content from unauthorized AI training and scraping.

## Features

- **Robots.xt Integration**: Easily generate and maintain comprehensive rules to block known AI user-agents.
- **Server-Side Blocking**: Rules for Nginx, Apache, and Cloudflare to drop requests from AI crawlers at the network edge.
- **HTML Meta Tag Injection**: Automated script to inject `noai`, `noimageai` and standard meta tags into your web pages.
- **Dynamic User-Agent Verification**: Detect and block crawlers trying to spoof regular browsers.
